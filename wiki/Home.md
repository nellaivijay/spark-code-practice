# Spark Code Practice - Home

Welcome to the Spark Code Practice wiki! This educational resource provides comprehensive hands-on labs for learning Apache Spark.

## Overview

Spark Code Practice is an independent educational resource designed to help data engineers, developers, and students practice and improve their Apache Spark skills through hands-on coding labs and exercises.

## What You'll Learn

This repository covers the complete Spark ecosystem:

### Core Concepts
- Spark architecture and fundamentals
- DataFrame operations and transformations
- Spark SQL and query optimization
- Data loading from various sources

### Advanced Topics
- Advanced DataFrame operations (joins, window functions)
- Complex data types (arrays, maps, structs)
- User Defined Functions (UDFs)
- Performance optimization techniques

### Streaming
- Structured Streaming fundamentals
- Windowed operations
- Event-time processing
- Watermarking and late data handling

### Machine Learning
- MLlib for machine learning
- Classification models
- Regression models
- Clustering algorithms
- Model evaluation

### Production Deployment
- Docker deployment
- Kubernetes deployment
- Monitoring and logging
- Security configuration
- CI/CD pipelines

## Getting Started

### Prerequisites

- Docker and Docker Compose
- Basic knowledge of Python or Scala
- Understanding of data structures
- Terminal/command line familiarity

### Quick Start

1. Clone the repository:
```bash
git clone https://github.com/nellaivijay/spark-code-practice.git
cd spark-code-practice
```

2. Run the setup script:
```bash
./scripts/setup.sh
```

3. Access Jupyter Notebook:
```
http://localhost:8888
```

4. Start with Chapter 1: Spark Fundamentals

## Lab Structure

The repository is organized into 7 comprehensive chapters:

1. **Chapter 1: Spark Fundamentals** - Introduction to Spark architecture and basic operations
2. **Chapter 2: Data Loading and Transformation** - Working with various data formats
3. **Chapter 3: Advanced DataFrame Operations** - Joins, window functions, complex types
4. **Chapter 4: Spark SQL and Optimization** - SQL queries and performance tuning
5. **Chapter 5: Spark Streaming Fundamentals** - Real-time data processing
6. **Chapter 6: Machine Learning with MLlib** - Building ML pipelines
7. **Chapter 7: Production Deployment** - Deploying Spark applications
